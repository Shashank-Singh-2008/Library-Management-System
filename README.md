 📚 Library Management System (Python)

A simple console-based Library Management System developed in Python. This project allows users to manage books by adding, viewing, searching, borrowing, returning, and deleting books.

 🚀 Features

- ➕ Add a new book
- 📖 View all books
- 🔍 Search for a book by title
- 📥 Borrow a book
- 📤 Return a book
- 🗑️ Delete a book
- 🚪 Exit the application

 🛠️ Technologies Used

- Python 3
- Functions
- Lists
- Dictionaries
- Loops
- Conditional Statements

 📂 Project Structure

```
Library-Management-System/
│
├── main.py        # Main program
├── library.py     # Library functions
└── README.md      # Project documentation
```

 ▶️ How to Run

1. Make sure Python 3 is installed.
2. Download or clone this repository.
3. Open the project folder in VS Code or any Python IDE.
4. Run the following command:

```bash
python main.py
```
